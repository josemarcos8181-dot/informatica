O que é Hardware?

Hardware é toda a parte física do computador, como monitor, teclado, mouse, placa-mãe, memória RAM e HD/SSD.

O que é Software?

Software é a parte lógica do computador: programas, aplicativos e sistemas operacionais como Windows, Linux e Android.

O que é Rede de Computadores?

É a conexão entre computadores para compartilhar informações, como internet, Wi-Fi ou rede local.
